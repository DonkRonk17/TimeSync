# TimeSync Branding Prompts

## Design System: Beacon HQ Visual System v1

**Core Principles:**
- Clean, modern, professional
- Time/synchronization theme
- Color palette: Deep blues (#1a237e), teals (#00bcd4), golds (#ffd700)
- Minimalist but distinctive

---

## 1. TITLE CARD (16:9 landscape)

**DALL-E Prompt:**
```
Create a professional title card for "TimeSync" - a universal time synchronization system.

Visual style: Clean, modern, tech-focused with subtle clock/time elements. Use dark background (#0a0a1a) with vibrant accents (teal #00bcd4, gold #ffd700).

Main elements:
- Bold title "TimeSync" in modern sans-serif font
- Tagline: "Universal Time Synchronization" in smaller text
- Abstract geometric pattern with clock hands, time zones, or sync symbols
- Subtle sacred geometry elements (dodecahedrons)
- Professional tech aesthetic

Layout: Horizontal 16:9 format, title center-left, time/sync visuals on right.

Style: Professional, minimal, futuristic. Think "enterprise software meets elegant timepiece design."
```

---

## 2. LOGO MARK (3:1 horizontal)

**DALL-E Prompt:**
```
Create a horizontal logo (3:1 aspect ratio) for "TimeSync" time synchronization software.

Visual style: Minimalist, professional, time-focused. Dark background with vibrant accents.

Elements:
- Left: Abstract icon with clock/sync symbol in geometric shapes
- Right: "TimeSync" text in clean modern font
- Accent colors: Teal (#00bcd4) and gold (#ffd700)
- Background: Deep space blue (#1a237e)

Style: Clean, scalable, professional. Suitable for GitHub headers.
```

---

## 3. APP ICON (1:1 square)

**DALL-E Prompt:**
```
Create a square app icon (1:1) for "TimeSync" time synchronization tool.

Visual style: Bold, simple, instantly recognizable at small sizes.

Elements:
- Central clock/sync symbol in geometric form
- Abstract design with circular time motif
- 2-3 colors: Teal (#00bcd4), gold (#ffd700), dark background
- No text (icon only)
- Clean lines, high contrast

Style: Modern, minimal. Must work at favicon size (64x64).
```

---

## Design Notes

**Tool Concept:** Universal timer that keeps all AI agents synchronized

**Visual Metaphors:**
- Clock face: Represents time tracking
- Sync arrows: Represents synchronization across systems
- Universal symbol: One timer for all

**Color Psychology:**
- Teal: Precision, technology, clarity
- Gold: Value, accuracy, importance
- Dark blue: Stable, trustworthy, professional

**Target Aesthetic:** Precise, synchronized, universal - like an atomic clock meets modern UI
